# heataxis

Thermal heat-load indices and physiological threshold detection for dairy-cattle
heat-stress analysis.

```{toctree}
:maxdepth: 2
:caption: Contents

api
```

## Overview

- **`heataxis.indices`** — the x-axis: cattle thermal indices, grouped by
  heat-exchange pathway.
- **`heataxis.thresholds`** — the y-axis: methods that locate the response
  threshold.
- **`heataxis.viz`** — shared, publication-ready plotting helpers.

See the [README](https://github.com/zerotonin/heataxis) for installation and a
quickstart.
